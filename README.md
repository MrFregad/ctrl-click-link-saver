# 🔗 Ctrl+Click Link Saver

Расширение для Chrome/Edge, которое **перехватывает Ctrl+Click на ссылках** и сохраняет URL в текстовый файл — вместо того чтобы открывать новую вкладку.

Удобно для сбора ссылок во время серфинга, ресёрча, или работы со списками статей.

---

## ✨ Возможности

- **Перехват Ctrl+Click** — ссылка не открывается, а сохраняется
- **Счётчик на иконке** — сразу видно сколько ссылок накоплено
- **Список в попапе** — все ссылки с заголовком, URL и временем
- **Скачать .txt** — выгрузка всего списка одной кнопкой
- **Защита от дублей** — одна и та же ссылка не добавится дважды
- **Клик в попапе** — можно открыть любую сохранённую ссылку

---

## 📁 Структура файлов

```
ctrl-click-link-saver/
├── manifest.json     # Конфиг расширения (Manifest V3)
├── content.js        # Перехватчик кликов на всех страницах
├── background.js     # Service worker: хранение и экспорт ссылок
├── popup.html        # UI попапа
├── popup.js          # Логика попапа
├── icon16.png        # Иконка 16x16
└── icon48.png        # Иконка 48x48
```

## 🚀 Установка

1. Скачай или клонируй репозиторий
2. Открой Chrome/Edge и перейди на страницу `chrome://extensions/`
3. Включи **Режим разработчика** (правый верхний угол)
4. Нажми **"Загрузить распакованное расширение"**
5. Выбери папку с файлами расширения

---

## 🖱️ Использование

| Действие | Результат |
|---|---|
| `Ctrl + Click` на ссылке | URL сохраняется, вкладка не открывается |
| Клик на иконку расширения | Открывается список сохранённых ссылок |
| Клик на ссылку в попапе | Открывает её в новой вкладке |
| Кнопка "Скачать .txt" | Скачивает все ссылки в файл |
| Кнопка "Очистить" | Удаляет весь список |

---

## 🔧 Технологии

- **Manifest V3** (актуальный стандарт Chrome Extensions)
- `chrome.storage.local` — хранение ссылок между сессиями
- `chrome.downloads` — скачивание файла из попапа
- `chrome.notifications` — уведомление при сохранении ссылки
